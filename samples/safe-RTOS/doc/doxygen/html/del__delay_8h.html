Function Documentation

void del_delayMicroseconds ( unsigned int  tiCpuInUs)

This function does nothing but consumes an exactely determined amount of CPU execution time before it returns. "Consumption of CPU time" means that not the elapsed world time counts; all interrupts etc. which interrupt the execution of this function won't be accounted. The blocking time of this function until return therefore is greater or equal to tiCpuInUs Microseconds.

Parameters
tiCpuInUsThe CPU executes further useless code this number of Microseconds.
