#define RTOS_SYSCALL_SUSPEND_ALL_TASKS_BY_PRIORITY   1

Index of implemented system call for raising a user context's current priority. The system call is wrapped in the inline function rtos_suspendAllTasksByPriority(). Please refer to this function for details.

#define RTOS_SYSCALL_RESUME_ALL_TASKS_BY_PRIORITY   2

Index of implemented system call for lowering a user context's current priority. The system call is wrapped in the inline function rtos_resumeAllTasksByPriority(). Please refer to this function for details.
