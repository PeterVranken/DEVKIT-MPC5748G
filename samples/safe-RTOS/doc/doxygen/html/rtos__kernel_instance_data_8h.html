Function Documentation

static rtos_kernelInstanceData_t* rtos_osGetInstancePtr ( void  )
inlinestatic

All data in use by the kernel is bundled in a large struct. If the RTOS is run on an MCU with several cores then each core uses its own dedicated instance of this struct. This function returns the struct to use by reference.

Returns
Get the instance pointer to the RTOS data set.
Remarks
This function my be called from OS contexts only. Any attempt to call it from a user task will cause a privileged exception. Use rtos_getInstancePtr() instead.
