Detailed Description

Specification of one kind-of-event instance of the binary search handle map. Used (only) for creation of the map object using mci_createMapBinarySearchCanId().

◆ doBinarySearchMapping

bool mci_specMapInstance_t::doBinarySearchMapping

Does the given kind-of-event require binary search of a CAN ID or is simple mapping sufficient? If doBinarySearchMapping is false then all other fields are meaningless and should be set to NULL or 0.

◆ mapStdCanIdToEdeIdxAry

const mci_uintMapValue_t* mci_specMapInstance_t::mapStdCanIdToEdeIdxAry

The lookup table for standard CAN IDs by reference or NULL if all CAN IDs are handled via binary search. Index into the table is the 11 Bit CAN ID, so it has 2048 entries. Array entry is the associated zero based index of the message in the event dispatcher system.

◆ mapExtCanIdToEdeIdxAry

The lookup table for extended CAN IDs by reference (and for standard IDs, too, if mapStdCanIdToEdeIdxAry is NULL), or NULL if the map instance is of kind "simple mapping".
