Detailed Description

◆ mci_uintMapValue_t

typedef uint8_t mci_uintMapValue_t

The definitions below depend on whether or not extended CAN IDs are in use. This needs to be decided before this header is included. Set MCI_SUPPORT_EXTENDED_CAN_IDS to 0 or 1 prior to including this file. The map value type. Normally, a uint8 will be appropriate but if more than 254 CAN messages are defined in at least one involved event dispatcher system then this type would overflow and a unit16_t becomes the better choice.
