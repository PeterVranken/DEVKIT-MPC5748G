      • createComparatorString

        public static SortOrder.Comparator<java.lang.String> createComparatorString(SortOrder.Order sortOrder)
        A comparator for sorting String objects is created and returned.
        Parameters:
        sortOrder - The comparator will support the given sort order. The following sort orders are supported:

        • lexical: The case insensitive sort order of Java String
        • ASCII: The natural sort order of Java String
        • numerical: The String contents are interpreted as numbers and the numbers' values are compared. See SortOrder.Order for additional details.
        • undefined: All compared elements will be treated as equal. Sorting will not change the order of elements
        Returns:
        Get a temporary comparator object as usable with the Java Collections class.
